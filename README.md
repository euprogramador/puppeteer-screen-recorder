# puppeteer-screen-recorder

Demonstra como usar puppeteer para gravar navegação de teste

Para rodar é necessário ter o ffmpeg instalado e o Xvfb.


basta executar o run-test.sh que será iniciado:

servidor xvfb 
gravação via ffmpeg do xvfb
execução do teste


será gerado um arquivo out.mp4 que representa a gravação do teste.
